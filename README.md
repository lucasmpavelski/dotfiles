## Setup

- Machine: `thinkpad x220`
- Hostname: `ronin`
- Operating System: `arch`
- Window Manager: `dwm`
- Shell: `zsh`
- Terminal Emulator: `st`
- File Manager: `lf, pcmanfm`
- Application Launcher: `dmenu`
- Compositor: `picom`
- Bar: `dwmblocks`
- Notification: `dunst`
- Text Editor: `nvim, nano`
